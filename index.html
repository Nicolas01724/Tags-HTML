Tags HTML


Primeiras Tags


  • HTML: Serve para informar ao navegador qu tipo de estrutura de dados está sendo construída, tendo links, imagens, etc...
  • Head: A tag head serve para providenciar informações gerais, configurações sobre o documento como seu título e coisas do tipo.
  • Body: A tag body serve para criar um corpo para o documento, tudo que estiver dentro dele será mostrado de alguma forma na página.
  • Header: A tag header server para criar o cabeçalho do site.
  • Section: Serve para separar a informação do site, agrupando o compeúto e separando a informação em áreas diferentes.
  • Div: A tag div serve também para agrupar e separar os conteúdos do site porém sem um estilo visual padrão.
  • Footer: A tag footer serve para criarmos um rodapé para o nosso site.
  • H1 até h6: As tags h1 até h6 servem para criar criar títulos e subtítulos para a página ajudando a indicar a importância de cada parte do conteúdo do site.
  • H7: A tag h7 não existe por si só no html, sendo assim o navegador não a reconhece como uma divisão para título ou subtítulo e acaba usando ela como um parágrafo normal.
  • Hr: A tag hr serve para criar uma linha no site.
  • P: A tag p serve para criar um parágrafo.
  • Ul: A tag ul serve para se criar um lista desordenada, ou seja não importa a ordem dos itens da lista que você está criando.
  • Li: A tag li serve para representar um item que faz parte de uma lista.
  • Br: A tag br serve para quebrar as linhas em um texto html.


Outras tags


  • A: A tag a serve para criar um link de outro site dentro do seu site ou colocar uma seção do seu site.
  • Strong: A tag strong serve para colocar o texto dentro dele em negrito.
  • Img: A tag img serve para colocar uma imagem na página.
  • Doctype: Declara que versão do html será usada na página.
  • Meta: a tag meta fornece informações sobre a página, como autor e coisas do tipo.
  • Title: A tag title define um título para a página.
  • Button: A tag button cria um botão clicável na sua página, podendo ser configurado para realizar ações.
  • Audio: A tag audio serve para inserir um arquivo de áudio na página.
  • Video: A tag video serve para inserir um vídeo na página.
  • Input: A tag input serve para receber os dados que o usuário digitar.
